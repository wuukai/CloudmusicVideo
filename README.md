### 使用方法： 
### 1.注册百度开发者平台账号，添加一个语音识别的应用程序，获取api_key和api_secret添加至api_baidu.py的'#############'
### 2.下载视频一段，下载音频一段，放置于同目录。
### 3.修改main.py  comment_id为网易云评论ID，video_file为视频路径，music_file为音频路径。
### 4.使用为 cmd cd 到本文件夹，python3 mian.py 即可
